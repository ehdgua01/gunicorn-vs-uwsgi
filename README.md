Gunicorn vs UWSGI

# 측정 요소

- CPU 사용량

  - 동시 요청 수를 증가시키며 CPU 사용량을 측정한다.
  - 요청 중 측정한 수치들의 평균값으로 측정

- 속도(latency)

  - 동시 요청 수에 따라 달라지는 요청부터 응답까지 걸리는 시간을 측정한다.
  - 마지막 응답 시간을 측정

- 메모리(RAM) 사용량

  - 동시 요청 수를 증가시키며 RAM 사용량을 측정한다.
  - 요청 중 측정한 수치들의 평균값으로 측정
